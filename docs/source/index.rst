CypherCat
========

**CypherCat is a Python package for testing methods of adversarial attacks and defenses of ML models**

CypherCat provides an secure-ML toolkit for members of all scientific disciplines with a simple, user-friendly API. Implementing a test attack and defense with CypherCat only takes a few lines of code.

    # Perform and evaluate defense
